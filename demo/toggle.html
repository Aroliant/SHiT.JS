SHiT.JS - Toggle()

Syntax

shit.toggle('#id name');		// use # to for div
shit.toggle('.class name');	// use . for class
shit.toggle('tag name');		// tag

Example

	<button onclick="shit.toggle('#container')">Toggle using ID</button>
	<button onclick="shit.toggle('.hello')">Toggle using Class</button>
	<button onclick="shit.toggle('div')">Toggle using Tag</button>
