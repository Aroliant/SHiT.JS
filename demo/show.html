SHiT.JS - Show()

Syntax

shit.show('#id name');		// use # to for div
shit.show('.class name');	// use . for class
shit.show('tag name');		// tag

Example

	<button onclick="shit.show('#container')">Show using ID</button>
	<button onclick="shit.show('.hello')">Show using Class</button>
	<button onclick="shit.show('div')">Show using Tag</button>
