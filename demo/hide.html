SHiT.JS - hide()

Syntax

shit.hide('#id name');		// use # to for div
shit.hide('.class name');	// use . for class
shit.hide('tag name');		// tag

Example

	<button onclick="shit.hide('#container')">Hide using ID</button>
	<button onclick="shit.hide('.hello')">Hide using Class</button>
	<button onclick="shit.hide('div')">Hide using Tag</button>
